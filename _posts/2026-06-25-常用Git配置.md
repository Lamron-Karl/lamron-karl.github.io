#### ~/.gitconfig

```ini
[user]
        name = weizean
        email = weizean@huawei.com

[core]
        editor = vim
        excludesFile = ~/.gitignore_global

[alias]
        ll = log --oneline --decorate
        st = status
        cp = cherry-pick
        co = checkout
        br = branch

[http]
        sslVerify = false
        postBuffer = 524288000
        version = HTTP/1.1

[http "https://github.com"]
        proxy = http://<user>:<password>@proxyuk.huawei.com:8080/
        sslVerify = false

[https "https://github.com"]
        proxy = https://<user>:<password>@proxyuk.huawei.com:8080/
        sslVerify = false
```

#### 配置说明

**[user]** — 用户信息

| 配置项 | 说明 |
|--------|------|
| `name` | 提交时显示的用户名 |
| `email` | 提交时关联的邮箱地址 |

**[core]** — 核心行为

| 配置项 | 说明 |
|--------|------|
| `editor` | 默认编辑器，用于编写 commit message 等 |
| `excludesFile` | 全局 `.gitignore` 文件路径，对所有仓库生效 |

**[alias]** — 常用别名

| 别名 | 等价命令 | 说明 |
|------|----------|------|
| `ll` | `log --oneline --decorate` | 简洁带装饰的提交历史 |
| `st` | `status` | 查看工作区状态 |
| `cp` | `cherry-pick` | 摘取指定提交 |
| `co` | `checkout` | 切换分支或恢复文件 |
| `br` | `branch` | 分支管理 |

**[http]** — HTTP 全局设置

| 配置项 | 说明 |
|--------|------|
| `sslVerify` | 是否校验 SSL 证书，内网环境可设为 `false` |
| `postBuffer` | 推送缓冲区大小（字节），调大可解决大仓库推送失败问题 |
| `version` | 强制使用 HTTP/1.1，解决某些代理不支持 HTTP/2 的问题 |

**[http/https "url"]** — 针对特定域名的代理设置

针对 `https://github.com` 单独配置 HTTP/HTTPS 代理，适用于需要通过公司代理访问外网的场景。

#### ~/.gitignore_global

全局忽略文件，对本机所有 Git 仓库生效，无需在每个项目中重复配置：

```
.vscode
.history
.codex
```

| 条目 | 说明 |
|------|------|
| `.vscode` | VSCode 编辑器本地配置目录 |
| `.history` | Local History 插件生成的历史记录目录 |
| `.codex` | Codex/Copilot 相关本地缓存目录 |

#### 常用配置命令

```bash
# 查看所有配置
git config --list

# 设置用户信息（全局）
git config --global user.name "Your Name"
git config --global user.email "you@example.com"

# 添加别名
git config --global alias.ll "log --oneline --decorate"

# 配置代理
git config --global http.proxy http://<user>:<password>@proxy.example.com:8080

# 取消代理
git config --global --unset http.proxy

# 关闭 SSL 验证（不推荐在公共网络使用）
git config --global http.sslVerify false

# 增大推送缓冲区（解决大文件推送超时）
git config --global http.postBuffer 524288000
```
