#### 常用软件

- **7-Zip**: <https://www.7-zip.org/>
  - 开源免费的压缩/解压软件，支持 7z、zip、tar 等多种格式，压缩率高
- **Beyond Compare**: <https://www.scootersoftware.com/>
  - 专业的文件与目录对比工具，支持文本、二进制、图片等多种格式的差异比对与合并
- **BongoCat**: <https://github.com/kuroni/bongocat-osu>
  - 可爱的桌面小猫咪，会跟随鼠标和键盘操作做出对应动作
- **Cherry Studio**: <https://cherry-ai.com/>
  - 支持多平台的 AI 桌面客户端，可聚合接入多种大语言模型
- **亿图图示（Edraw）**: <https://www.edrawsoft.cn/>
  - 国产跨平台绘图软件，支持流程图、思维导图、UML 等多种图表类型
- **Everything**: <https://www.voidtools.com/>
  - Windows 下极速文件搜索工具，索引本地文件，搜索结果几乎即时呈现
- **Git**: <https://git-scm.com/>
  - 分布式版本控制系统，现代软件开发的必备工具
- **VS Code**: <https://code.visualstudio.com/>
  - 微软出品的免费开源代码编辑器，插件生态丰富，支持几乎所有主流语言
- **MindMaster**: <https://www.edrawsoft.cn/mindmaster/>
  - 亿图旗下的专业思维导图软件，支持多种导图风格与协作功能
- **Notepad++**: <https://notepad-plus-plus.org/>
  - Windows 下轻量级开源文本编辑器，支持语法高亮与多标签编辑
- **PixPin**: <https://pixpinapp.com/>
  - 功能强大的截图/贴图工具，支持长截图、OCR、标注等功能
- **Sublime Text**: <https://www.sublimetext.com/>
  - 轻量快速的跨平台文本编辑器，以流畅的操作体验著称
- **Tabby**: <https://tabby.sh/>
  - 开源跨平台终端模拟器，支持 SSH、串口、本地 Shell，界面美观，高度可定制
- **Wireshark**: <https://www.wireshark.org/>
  - 开源网络协议分析工具，可实时捕获并详细分析网络数据包
- **Xshell**: <https://www.netsarang.com/zh/xshell/>
  - 功能强大的 SSH 客户端，常用于远程连接和管理 Linux 服务器

#### 常用 VSCode 插件

- **C/C++**: <https://marketplace.visualstudio.com/items?itemName=ms-vscode.cpptools>
  - 微软官方 C/C++ 支持，提供代码补全、调试、IntelliSense 等功能
- **Python**: <https://marketplace.visualstudio.com/items?itemName=ms-python.python>
  - 微软官方 Python 支持，提供代码补全、调试、Lint、环境管理等功能
- **Project Manager**: <https://marketplace.visualstudio.com/items?itemName=alefragnani.project-manager>
  - 快速切换和管理多个项目，支持收藏常用工作区
- **Better Comments**: <https://marketplace.visualstudio.com/items?itemName=aaron-bond.better-comments>
  - 对注释进行颜色分类，让 TODO、警告、说明等注释一目了然
- **Bookmarks**: <https://marketplace.visualstudio.com/items?itemName=alefragnani.bookmarks>
  - 在代码中添加书签，方便在多处位置之间快速跳转
- **Todo Tree**: <https://marketplace.visualstudio.com/items?itemName=gruntfuggly.todo-tree>
  - 在侧边栏以树状结构展示代码中所有的 TODO、FIXME 等标记
- **vscode-icons**: <https://marketplace.visualstudio.com/items?itemName=vscode-icons-team.vscode-icons>
  - 为文件资源管理器中的文件和文件夹提供丰富的图标，提升可读性
- **GitLens**: <https://marketplace.visualstudio.com/items?itemName=eamodio.gitlens>
  - 增强 VSCode 内置 Git 功能，可在行内显示提交记录、作者信息，支持历史对比
- **Prettier**: <https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode>
  - 主流的代码格式化工具，支持 JS、TS、CSS、JSON、Markdown 等多种语言
- **Markdown All in One**: <https://marketplace.visualstudio.com/items?itemName=yzhang.markdown-all-in-one>
  - Markdown 写作增强，提供快捷键、自动补全、目录生成、预览等功能
- **Codex**: <https://marketplace.visualstudio.com/items?itemName=GitHub.copilot>
  - GitHub Copilot 驱动的 AI 代码补全工具，可根据上下文生成代码片段
- **WSL**: <https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-wsl>
  - 在 Windows 上通过 WSL 直接打开并编辑 Linux 环境中的文件和项目
- **Zhihu Fisher**: <https://marketplace.visualstudio.com/items?itemName=shalldie.zhihu>
  - 在 VSCode 中浏览知乎内容，摸鱼神器
- **Thief-Book**: <https://marketplace.visualstudio.com/items?itemName=cweijan.thief-book>
  - 将小说/电子书内容以状态栏文字的形式低调显示，方便工作中偷偷阅读
